# UI Component Tests

Тесты для общих UI компонентов и примитивов.

## Компоненты в этой области:
- Badge - компонент для отображения меток и статусов
- Button - кнопки различных типов и размеров
- Card - контейнеры для группировки контента
- Input - поля ввода различных типов
- InteractiveTag - интерактивные теги с возможностью удаления

## Запуск тестов:
```bash
npm run test:component -- --spec 'cypress/component/ui/**/*.cy.tsx'
```
