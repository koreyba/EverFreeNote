# Testing Strategy

## Test Coverage Goals
- Manual testing of swipe visibility.
- Manual testing of confirmation dialog.
- Verification of deletion in Supabase dashboard.

## Manual Testing
- [x] Swipe note in list: Red "Delete" button appears.
- [x] Tap "Delete" button: Note disappears immediately.
- [x] Open note in editor: Tap trash icon.
- [x] Confirmed: Editor closes, note disappears immediately.
