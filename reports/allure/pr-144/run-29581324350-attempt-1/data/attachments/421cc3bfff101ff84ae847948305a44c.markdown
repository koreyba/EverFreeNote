## 🛑 [color-contrast] Elements must meet minimum color contrast ratio thresholds
- **Severity**: serious
- **Description**: Ensure the contrast between foreground and background colors meets WCAG 2 AA minimum contrast ratio thresholds
- **Learn More**: [Deque University Link](https://dequeuniversity.com/rules/axe/4.12/color-contrast?application=playwright)

### Affected Elements:

#### Element 1
- **Selector**: `.select-none > h2`
- **HTML Snippet**:
  ```html

  ```
- **How to Fix**:

  >   Element has insufficient color contrast of 1.85 (foreground color: #464b50, background color: #1b2026, font size: 15.0pt (20px), font weight: normal). Expected contrast ratio of 4.5:1
